# Learning Analytics Dashboard

A single-page frontend dashboard built using React and modern charting libraries.

## Features
- KPI Summary Cards
- Course Progress Bar Chart
- Pass Percentage Chart
- Assessment Completion Donut Chart
- Grade Breakdown Pie Chart
- District Ranking Section
- Light & Dark Theme with persistence
- Mock API using local JSON data (2024 & 2025)

## Tech Stack
- React
- TypeScript / JavaScript
- Recharts
- CSS / Tailwind / Styled Components

## How to Run
```bash
npm install
npm run dev
